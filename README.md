# 🎁 Give-What-You-No-Longer-Use

**Una plataforma solidaria para donar lo que ya no usás.**  
Promovemos la economía circular, la sostenibilidad y el impacto comunitario.  
¡Doná objetos y ayudá a quien más lo necesita!

---

## 🌍 ¿Qué es esto?

> **Give-What-You-No-Longer-Use** es una plataforma de clasificados sociales donde los usuarios pueden donar artículos de segunda mano como ropa, muebles, libros y más.

Este proyecto está potenciado por IA para mejorar el matching entre donantes y receptores, y está construido con tecnologías modernas como:

- 🐍 **Python** + **FastAPI**
- ☁️ **Azure**
- ⚙️ **Docker**
- 🧠 **AI-powered recommendations**
- 🌐 **Frontend y Backend integrados**

---

## 🛠️ Estructura del Proyecto

backend/ → Lógica del servidor (FastAPI, controladores, rutas, etc.)
frontend/ → Interfaz de usuario (HTML/CSS/JS o framework web)
scripts/ → Scripts de automatización (sincronización, mantenimiento)
voice-ai-agent/ → Módulo experimental con IA para recomendaciones inteligentes

---

## 🚀 Primeros pasos

1. **Cloná el repositorio:**

```bash
git clone https://github.com/miguel-agui/Give-What-You-No-Longer-Use.git
cd Give-What-You-No-Longer-Use
Configurá el entorno local:

Python 3.11+

Docker (opcional, para entornos reproducibles)

FastAPI

Frontend configurado según stack (ej: React, Vue, HTML/CSS puro)

Ejecutá el backend localmente:

bash
Copiar código
cd backend
uvicorn main:app --reload
Ejecutá el frontend:

bash
Copiar código
cd frontend
# Comandos según framework (npm run dev, live-server, etc.)
